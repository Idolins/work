# 人员管理系统


# 所用技术

- 前端采用 Bootstrap

- 后台使用 Struts2

- 持久层使用 Hibernate

- 语言kotlin

# 功能

- [x] 登陆
- [x] 注册
- [x] 文件的上传下载
- [ ] 批量下载和上传
- [x] 地理位置签到
- [ ] 公告/通知
- [ ] 站内信
- [ ] 人员查询
